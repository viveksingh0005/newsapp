# 📰 Real-Time News & Market Dashboard

A modern **React-based News Dashboard** that provides real-time news updates across multiple categories such as **Cricket, Trading, Gold & Commodities, International News, Entertainment, and Top Headlines**.

This project integrates external APIs to fetch live news data and presents it in a clean, responsive interface built with **React and Tailwind CSS**.

---

## 🚀 Features

* 📰 **Top Headlines** – Latest trending news
* 🏏 **Cricket Section** – Cricket news and updates
* 📈 **Trading News** – Stock market and finance updates
* 🪙 **Gold & Commodities** – News related to metals and commodities
* 🌍 **International News** – Global headlines
* 🎬 **Entertainment** – Movies, celebrities, and entertainment industry news
* 🔍 **External Article Links** – Open full article on original website
* 📱 **Responsive Design** – Works on desktop, tablet, and mobile
* ⚡ **Fast Loading UI** – Built with Vite for fast performance

## ⚙️ Installation

Clone the repository:

```bash
git clone https://github.com/viveksingh0005/newsapp.git
```

Go to project directory:

```bash
cd news-dashboard
```

Install dependencies:

```bash
npm install
```

Run the development server:

```bash
npm run dev
```
